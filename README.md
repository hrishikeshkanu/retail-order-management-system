A comprehensive, full-stack order and inventory tracking platform designed for retail businesses to bridge the gap between suppliers and customers. This system replaces manual, error-prone ordering with an automated, secure, and user-friendly digital interface. 

Developed as an academic project at the **Institute of Information Technology (IIT), Jahangirnagar University**.

## ✨ Key Features

* **🔐 Secure Authentication:** JWT (JSON Web Tokens) for stateless session management and Bcrypt for secure password hashing.
* **🛍️ Dynamic Product Catalog:** Real-time fetching of available products from the database, preventing out-of-stock orders.
* **🧮 Automated Financial Logic:** Server-side calculation of order totals (`Price × Quantity`) to eliminate human error and client-side manipulation.
* **📊 Real-Time Order History:** Immediate visual feedback and persistent tracking of customer transactions.
* **🎨 Modern UI/UX:** Responsive Single-Page Application (SPA) feel using the vanilla Fetch API and a sleek "Glassmorphism" design aesthetic.

## 🛠️ Tech Stack

**Frontend:**
* HTML5, CSS3, Vanilla JavaScript (ES6+)
* Fetch API for asynchronous requests
* Glassmorphism UI Design

**Backend:**
* Node.js (Runtime)
* Express.js (RESTful API Framework)
* JSON Web Tokens (JWT) & Bcrypt (Security)

**Database:**
* MongoDB Atlas (Cloud NoSQL Database)
